== Description ==

[awe.sm](http://totally.awe.sm/) provides sharing analytics. This plugin provides awe.sm-powered buttons for Facebook, Twitter, Tumblr, LinkedIn, Google+, Pinterest, and email. It requires an awe.sm subscription for stats.

**Contributing code**

The development source code for this plugin is available on [awe.sm's GitHub account](https://github.com/awesm/wordpress-plugin). [Pull Requests](https://github.com/awesm/wordpress-plugin/pulls) and code discussion welcome.

== Installation ==

1. Install awe.sm for WordPress either via the WordPress.org plugin directory or by uploading the files to your server's `/wp-content/plugins/` directory.

2. After activating the plugin, you will need to specify your awe.sm project's API key in the plugin settings. You can find your awe.sm project API key in your [project settings](https://so.awe.sm/).

3. Customize which sharing buttons you want to display.
